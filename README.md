# Evidence Vault DGC

## Overview
This project is designed to manage evidence using a combination of FastAPI for the backend, PostgreSQL as the database, and MinIO for object storage.

## Quickstart
1. Clone the repo: `git clone https://github.com/NievaPablo1905/EvidenceVaultDGC.git`
2. Navigate to the project directory: `cd EvidenceVaultDGC`
3. Copy the example environment variables: `cp .env.example .env`
4. Start the services: `docker-compose up`
